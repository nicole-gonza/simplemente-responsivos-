@media queries

CSS use the @media rule to detect information about the device being used to access the webside. For example, it can be used to detect the size of the device,its capabilities, and its orientation. For this course,we will be using it to detect the device size, but keep in mind thar it es capable of much more.

Tablet:width betwenn 481px and 1023px
Phone: width less than or equal to 480px
